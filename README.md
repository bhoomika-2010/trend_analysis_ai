# Trend Analysis AI

EDAI Semester 5 Project

## Description
Trend Analysis AI is a project that analyzes trends across different platforms
using engagement, sentiment, and share of conversation to help understand
what topics are gaining traction.

## Features
- Platform-wise trend analysis
- Sentiment analysis
- Engagement comparison
- Interactive visualizations

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- Chart.js
- SQL Database

## How to Run
1. Clone the repository
2. Install required Python libraries
3. Run the backend server
4. Open the frontend in your browser
